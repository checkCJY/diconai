# 자가정검 체크리스트(디코나이)

## 체크 방법

- `[ ]` 미확인 / 미구현
- `[x]` 확인 완료 / 발표 가능

## 주의사항

- 체크한 항목은 발표에서 실제로 시연 가능해야 합니다.
- 코드가 있다가 아니라 실제로 동작한다를 기준으로 체크합니다.
- 4차 발표는 3차처럼 화면만 보이는 것이 아니라 **AI + 비동기 + 운영 + 모니터링 구조가 연결되어 있는지**를 기준으로 점검합니다.
