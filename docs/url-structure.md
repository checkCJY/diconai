# URL 구조

현재 정의된 URL 목록입니다. 프로젝트 진행에 따라 지속적으로 확장될 예정입니다.

---

## DRF Server (port 8000)

| Method | Path | 설명 |
|--------|------|------|
| GET | `/` | 대시보드 렌더링 — `dashboard.html` 템플릿 반환 (작동 확인용) |
| GET/POST | `/admin/` | Django Admin 패널 |

### 비고
- `/` 는 현재 로그인 없이 바로 대시보드를 렌더링하는 임시 뷰입니다.
- 추후 인증(로그인) 흐름이 추가되면 `/login/` → 인증 → `/dashboard/` 형태로 분리될 예정입니다.

---

## FastAPI Server (port 8001)

| Method | Path | 설명 |
|--------|------|------|
| GET | `/health/` | 서버 상태 확인 — `{"status": "ok"}` 반환 |

### 비고
- 현재는 헬스체크 엔드포인트만 존재합니다.
- 추후 AI 예측, 센서 데이터, 외부 시스템 연동 API 등이 이 서버에 추가될 예정입니다.
