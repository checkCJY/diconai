# 증빙자료 인덱스

본 폴더는 "IF 선정 부첨"(`전력AI-모델-의사결정-개발문서.html`)의 모든 주장을 검증할 수 있는 원본 자료 모음임. 부첨 문서가 인용하는 자료들의 마크다운 원본·raw 데이터를 도메인별로 정리했음.

전력 AI 본문 PDF와 알람 시스템 본문 PDF는 별도 부록으로 본 폴더에 함께 동봉됨. 두 본문 PDF의 §9.3 "근거문서" 목록이 가리키는 원본도 `power/`·`alarm/`에 모두 포함되어 있어, PDF의 모든 근거를 본 폴더에서 직접 확인할 수 있음(아래 §1-B 참조).

---

## 폴더 구조

```
증빙자료/
├── 00_README.md (본 파일)
├── power/       전력 IF 관련 (본문 SoT·설계 의사결정·임계치·더미·un-downgrade·5축)
├── alarm/       알람 결합 관련 (Track 2→T1 plan·AI mute·decide_alarm·combine_risk)
├── perf/        성능·검증 관련 (정확도·리드타임·DB폭주 사고·통합진단)
└── 공용/        도메인 무관 자료
    ├── STEP/    학습 자료 STEP 0~5 (가스·전력 공통 프레임워크)
    └── PoC/     채널 상관성 PoC 리포트 + raw CSV
```
