| **운영 데이터 ACF/FFT** | 일·주 주기성 강함 → SARIMA 도입 정당화 / 약함 → 휴리스틱 유지 |
| **야간 baseline 채널별** | 채널별 차이 큼 → baseline 학습 / 거의 균등 → 정격 30% 유지 |
| **운영자 false positive 빈도** | 높음 → 임계 보수화 / 낮음 → 현재 유지 |

### 4.3 의사결정 패턴 재확인

| 패턴 | 시연 전 단계 | 시연 후 단계 |
|---|---|---|
| 1차 PoC → 본격 확장 | 4채널 / watt 단독 / 휴리스틱 | 16채널 / A/V 다변량 / SARIMA |
| 한계 명시 + 보완 | ARIMA 단발 spike → IF + threshold | multi-step + CUSUM 추가 |
| 도메인 의존 | 전력 = un-downgrade / 가스 = 격하 | confusion matrix 측정 후 재검토 |
| 운영 데이터 검증 | dummy 검증 | 운영 1~2주 후 임계·가중치 튜닝 |
