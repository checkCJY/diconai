# 로그인 페이지 (`login.html`)

## 개요

산재 예방 통합 관제 플랫폼의 로그인 화면입니다.  
`/` 경로로 접근 시 렌더링됩니다.

---

## 화면 구성 요소

| 번호 | 요소 | 설명 |
|------|------|------|
| ① | 배경 | 파란색(`#1976d2`) 단색 배경 |
| ② | 로고 | 에어위드 로고 영역 (`<img>` 교체 또는 플레이스홀더) |
| ③ | 플랫폼명 | "산재 예방 통합 관제 플랫폼" 텍스트 |
| ④ | 입력란 | 아이디 / 비밀번호 / 비밀번호 확인 3개 필드 |
| ⑤ | 로그인 버튼 | 유효성 검사 통과 후 API 호출 |
| ⑥ | 문의 문구 | 로그인 관련 문의 안내 텍스트 |

---

## 입력 필드 공통 동작

- **비식별화**: 비밀번호 · 비밀번호 확인 필드는 `type="password"` 처리
- **지움 버튼**: 각 필드에 문자가 입력되는 즉시 활성화, 클릭 시 해당 필드 초기화 및 포커스 복귀
- **Enter 키**: 모든 필드에서 Enter 입력 시 로그인 버튼 동작 실행

---

## 유효성 검사

> 로그인 버튼 클릭 시 모든 필드를 순서대로 검사합니다.  
> 검사 실패 시 해당 필드 하단에 에러 메시지를 표시하고 제출을 중단합니다.

### 아이디

| 조건 | 메시지 |
|------|--------|
| 미입력 | 아이디를 입력해주세요. |
| 영문·숫자 이외의 문자 포함 | 아이디는 영문 또는 숫자만 입력할 수 있습니다. |
| 4자 미만 또는 20자 초과 | 아이디를 4~20자로 입력해주세요. |

### 비밀번호

| 조건 | 메시지 |
|------|--------|
| 미입력 | 비밀번호를 입력해주세요. |
| 8자 미만 | 비밀번호를 8자 이상 입력해야 합니다. |
| 영문·숫자·특수문자 중 1가지만 포함 | 비밀번호는 영문, 숫자, 특수문자 중 2가지 이상을 포함해야 합니다. |

### 비밀번호 확인

| 조건 | 메시지 |
|------|--------|
| 미입력 | 비밀번호를 입력해주세요. |
| 비밀번호 필드와 불일치 | 비밀번호가 일치하지 않습니다. |

---

## 로그인 처리 흐름

1. 모든 유효성 검사 통과
2. `POST /api/auth/login/` 호출 (JSON body: `{ username, password }`)
3. 응답 성공 → `/dashboard/` 리다이렉트
4. 응답 실패 → 아이디 필드에 `"아이디 또는 비밀번호가 올바르지 않습니다."` 표시
